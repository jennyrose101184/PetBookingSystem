üìã Quick Integration (3 Steps)

Step 1: Copy the Widget Files

Download these files from the deployment:

  • https://jaehubbookingsystem.netlify.app/booking-widget.js
  • https://jaehubbookingsystem.netlify.app/booking-widget.css

Step 2: Add to Your Website HTML

Method 1: Direct CDN Integration (Recommended)

<!-- Add this where you want the booking form to appear -->
<div id="pet-booking-widget"></div>

<!-- Add these before closing </body> tag -->
<link rel="stylesheet" href="https://jaehubbookingsystem.netlify.app/booking-widget.css">
<script src="https://jaehubbookingsystem.netlify.app/booking-widget.js"></script>
<script>
document.addEventListener('DOMContentLoaded', function() {
    BookingWidget.init('pet-booking-widget', {
        apiUrl: 'https://your-backend-api.com/api' // Replace with your API URL
    });
});
</script>
Method 2: Auto-Initialize (Simplest)

<!-- Add this where you want the booking form -->
<div id="booking-widget-auto"></div>

<!-- Add these before closing </body> tag -->
<link rel="stylesheet" href="https://jaehubbookingsystem.netlify.app/booking-widget.css">
<script src="https://jaehubbookingsystem.netlify.app/booking-widget.js"></script>

Step 3: Configure Your Backend API

You need to deploy the backend server from server/app.cjs to a hosting service like:

  • Heroku (recommended)
  • Railway
  • DigitalOcean
  • AWS/Azure
