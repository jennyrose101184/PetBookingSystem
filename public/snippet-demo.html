This page demonstrates how to integrate the booking widget into any website as a simple snippet.

Integration Method 1: Auto-Initialize

Simply add a div with id booking-widget-auto and include the JavaScript file:

<!-- Widget Container -->
<div id="booking-widget-auto"></div>

<!-- Widget Scripts -->
<link rel="stylesheet" href="./dist/booking-widget.css">
<script src="./dist/booking-widget.js"></script>

Integration Method 2: Manual Initialize

For more control, you can manually initialize the widget:

<div id="my-booking-widget"></div>

<script>
  BookingWidget.init('my-booking-widget', {
    apiUrl: 'https://your-api.com/api' // Optional
  });
</script>
